# web-file

Nuxt 4 portfolio project for local-first file tooling. It demonstrates browser-only image conversion/compression with WASM codecs, client-side PDF workflows, PWA installability, TypeScript, Tailwind CSS, ESLint, and i18n.

## Commands

```bash
pnpm install
pnpm dev
pnpm lint
pnpm typecheck
pnpm test
```
